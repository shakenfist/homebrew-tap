# Shaken Fist Homebrew Tap

Homebrew formulae for [Shaken Fist](https://github.com/shakenfist) projects.

## Installation

```bash
brew tap shakenfist/tap
```

Then install any available formula:

```bash
brew install ryll
```

Or install directly without adding the tap first:

```bash
brew install shakenfist/tap/ryll
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [ryll](https://github.com/shakenfist/ryll) | A Rust SPICE VDI test client |

## Platform Support

All formulae in this tap provide pre-built binaries for Apple Silicon
(aarch64) Macs only. Intel Macs are not supported.
